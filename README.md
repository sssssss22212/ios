# 🧱 Decorative Blocks — Minecraft Forge 1.16.5

Мод добавляет 15 красивых декоративных блоков с кастомными текстурами и отдельной вкладкой в Creative.

## Блоки

### 🏛️ Декоративные
| Блок | Описание |
|------|----------|
| Marble Column | Мраморная колонна |
| Marble Tiles | Мраморная плитка |
| Ornate Tiles | Узорная плитка (тёмная с золотом) |
| Sandstone Tiles | Плитка из песчаника |
| Dark Bricks | Тёмный кирпич |
| Carved Stone | Резной камень |

### 🔮 Витражное стекло
| Блок | Цвет |
|------|------|
| Blue Crystal Glass | Синее |
| Red Crystal Glass | Красное |
| Purple Crystal Glass | Фиолетовое |
| Golden Crystal Glass | Золотое |

### ⚙️ Металлические
| Блок | Описание |
|------|----------|
| Steel Block | Стальной блок |
| Copper Plating | Медная обшивка |
| Industrial Grate | Промышленная решётка (прозрачная) |
| Rusted Iron | Ржавое железо |
| Polished Steel | Полированная сталь |

## Сборка через GitHub Actions

1. Создай репозиторий на github.com
2. Загрузи все файлы этого проекта
3. GitHub автоматически начнёт сборку
4. Зайди во вкладку **Actions** → последний билд → **Artifacts** → скачай `decorativeblocks-1.16.5.zip`
5. Внутри будет `.jar` файл для папки `mods/`
